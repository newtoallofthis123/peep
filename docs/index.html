Personalized Environment and Execution Prompt

PEEP stands for Personalized Environment and Execution Prompt. Okay, full disclosure, I wasn't exactly sure what PEEP stands for, I just used ChatGPT, but I do know what it does. It is a command line tool that allows you to use quick commands quickly in the syntax of standard speech.

Installation

git clone https://github.com/newtoallofthis123/peep.git
cd peep
pip install -r requirements.txt
python src/main.py
